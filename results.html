Results

This page explores the effect that various techniques have on the error of the flower classification models. Error is defined as the sum of false positive and false negative classifications.

Our starting point is a set of photos containing 80 examples each of five flowers. For each flower we split the photo set in half to give 40 training photos and 40 test photos. We use the training photos to generate flower classification models, before testing the models on the test set.
